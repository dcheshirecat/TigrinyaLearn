The Ge'ez Fidel (ፊደል)
Tigrinya uses the ancient Ge'ez script — an abugida where each character is a consonant+vowel pair. 7 vowel orders · 22 consonant families. Tap any character to explore it and hear it pronounced.
Pick a character to trace

ሀ
How to write Ge'ezWritten left to right. Each fidel is a single character. The ghost shows the target — try to cover it with your strokes. Accuracy is measured by how well your strokes overlap the character.
