The Ge'ez Fidel (ፊደል)
Tigrinya uses the ancient Ge'ez script — one of the world's oldest writing systems. Each symbol is called a fidel. The script is an abugida: each character is a consonant + vowel pair. There are 7 vowel orders and ~22 consonant families. Tap any character to explore it.
