The Ge'ez Fidel (ፊደል)
Tigrinya uses the ancient Ge'ez script, one of the world's oldest writing systems. Each symbol is called a fidel (ፊደል). The script is an abugida — each character represents a consonant + vowel combination. There are 7 vowel orders, and ~33 consonant families, giving over 230 characters. Tap any character to learn it.

How to write Ge'ez Ge'ez is written left to right, like English. Each fidel is a single character — no separate joining needed. Focus on the overall shape first, then add the vowel marker (the small extension that changes the vowel sound).
